Pythagorean Tree

Example Pythagorean Tree

Everybody knows the Pythogorean theorem. If not or if you want more interesting information, look at Wikipedia: Pythagorean theorem [last seen 2023-09-22]. We draw squares over both of the short sides of a rectangular triangle. Then we take the sides of the squares opposite to the sides belonging to the triangle as new hypotenuses, and we construct two new smaller triangles. (In a rectangular triangle the side opposite to the right angle is called hypotenuse, see Hypotenuse - Wikipedia [last seen 2023-09-22].)



The procedure described can be continued, and this way we get a lot of rectangular triangles with with squares at their short sides. This can be repeated until a certain short length of the hypotenuse is reached. The drawing using turtlegraphics looks like some kind of an abstract tree. For more details and further information see Pythagorean Tree (agnesscott.org) [last seen 2023-09-22].



The code is written in Python 3.9. Function “tree” contains two recursive calls so that each triangle branches into two smaller triangles if possible.
