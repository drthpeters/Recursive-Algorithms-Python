Palindromes

Example Palindromes

A palindrome is a word or a group of words (or a number) that is the same when you read it in the normal direction from left to right or in the reverse direction. Simple examples are “noon” or “level”, see also PALINDROME | English meaning - Cambridge Dictionary [last seen 2023-09-24]. There is also a recursive definition, see Using recursion to determine whether a word is a palindrome (article) | Khan Academy [last seen 2023-09-24]. The version we apply in our Python script is:

  • A word consisting of at most one character is a palindrome.
  • A word is not a palindrome if the first and the last letter are different.
  • Otherwise remove the first and the last letter from the word and check the rest.

A short group of words like “no lemon no melon” is considered as a palindrome too. We want our program to detect such palindromes. So we must remove spaces from the group of words, and we must convert all letters to capitals. This is done in function checkPalindrome. From that function the recursive function isPalindrome is called. The script also contains a couple of examples of words, numbers and small groups of words, some of which are palindromes and some are not. If you want the program to detect palindromes as “Madam, I'm Adam”, you have to add a statement to function checkPalindrome.
